# Natural Language Processing — Hate-Speech Classification

A text-classification project that detects hate speech in text data.

## What it does
- `hate_speech.ipynb` — the full pipeline: text cleaning, tokenisation, feature extraction (vectorisation), model training, and evaluation (accuracy / precision / recall).
- `hate_speech.csv` — the labelled dataset.

## How it works
Raw text is normalised and converted into numerical features; a classifier is trained to distinguish hateful from non-hateful text; performance is measured on held-out data. The notebook walks through the practical NLP workflow end to end.

## Role within my work (MyOTGO)
Natural-language processing underpins the conversational AI in the **MyOTGO** ecosystem — the voice / chat assistant's intent understanding, command parsing and text reasoning. This project demonstrates the core text-classification and language-processing techniques behind those features.
